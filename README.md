# Prompt — AI-ассистент для браузера

Универсальное расширение для Chrome, которое анализирует содержимое любых веб-страниц с помощью AI-моделей через OpenAI-совместимый API. Выполняет чат с контекстом страницы, обрабатывает изображения и выдаёт краткие выжимки.

## Возможности

### Анализ веб-страниц
- Извлечение контента с любых сайтов — автоматически определяет основной текст
- Readability-эвристика — исключает рекламу, навигацию, сайдбары
- Приоритет выделения — если пользователь выделил текст, используется только он
- Краткая выжимка — AI summarization содержимого страницы

### Чат с AI
- Контекст текущей вкладки — галочка 📄 включает/выключает контекст страницы в чате
- Многоходовой диалог — история сохраняется, можно уточнять
- Управление токенами — автоматическая обрезка истории при приближении к лимиту
- Отправка по Ctrl+Enter — быстрый ввод

### Обработка изображений
- Контекстное меню — ПКМ на любом изображении → два варианта:
  - **«🖼️ Получить промт изображения»** — подробное описание изображения
  - **«🔤 Перевод текста с изображения»** — распознавание и перевод текста
- Отдельные промпты — независимая настройка для каждого типа анализа
- CORS fallback — обходит ограничения CORS через background service worker
- Multimodal запросы — поддержка vision-моделей

### Интерфейс
- Тёмная/Светлая тема с переключателем
- Настройка размера шрифта (8-16pt)
- Плавающие кнопки управления и настроек
- Блок размышлений модели (reasoning, сворачиваемый)
- Бейдж источника — всегда видно, с какой страницы контекст
- Счётчик времени генерации — показывает как долго ждём ответ
- Кнопки копирования в блоках кода — быстрое копирование одним кликом
- Умный автоскролл — как ChatGPT/Claude (кнопка «↓» при прокрутке вверх)

## Требования

### Обязательные
- Браузер: Chrome 114+ или совместимый (Edge, Brave)
- AI сервер: OpenAI-совместимый API (LM Studio, Ollama, OpenRouter и др.)
  - [LM Studio](https://lmstudio.ai/)
  - [Ollama](https://ollama.com/)

### Рекомендуемые модели
- Для текста: Qwen 2.5 7B-72B, Llama 3 8B-70B, Mistral
- Для изображений: Модели с поддержкой vision/multimodal (LLaVA, Qwen-VL и др.)
- VRAM: 8GB+ для локальных моделей

## Установка

### 1. Настройка AI-сервера (пример с LM Studio)

1. Скачайте и установите [LM Studio](https://lmstudio.ai/)
2. Загрузите модель
3. Запустите локальный сервер: вкладка "Local Server" → "Start Server"
4. По умолчанию: http://localhost:1234

### 2. Установка расширения

1. Откройте Chrome → chrome://extensions/
2. Включите "Режим разработчика" (справа вверху)
3. Нажмите "Загрузить распакованное"
4. Выберите папку с расширением
5. Иконка Prompt появится в панели расширений

### 3. Первая настройка

1. Откройте любой сайт
2. Кликните на иконку Prompt в панели браузера
3. Откроется боковая панель
4. Перейдите в настройки (икона ⚙️ справа вверху):
   - API URL: http://localhost:1234/v1
   - API Key: ваш ключ авторизации (если требуется)
   - Модель: название вашей модели (или включите "Использовать загруженную модель")

## Использование

### Анализ страницы
1. Откройте любой сайт
2. Кликните на иконку Prompt
3. Нажмите кнопку 🚀 (поле ввода пустое) → AI проанализирует страницу
4. Получите краткую выжимку

### Чат с AI
1. Откройте сайт который хотите изучить
2. Кликните на иконку Prompt
3. Нажмите кнопку 📄 (включить контекст страницы)
4. Введите запрос в поле ввода (например: "О чём эта статья?")
5. Нажмите Ctrl+Enter или кнопку 🚀
6. AI ответит с учётом контекста текущей страницы
7. Задавайте уточняющие вопросы — история сохранится

### Обработка изображения
1. ПКМ на любом изображении
2. Выберите нужный пункт:
   - «🖼️ Получить промт изображения» — для получения описания
   - «🔤 Перевод текста с изображения» — для OCR и перевода
3. Результат появится в sidebar

## Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| Ctrl+Enter | Отправить запрос / Анализ страницы / Стоп (во время генерации) |
| Ctrl+R     | Очистить чат и сбросить историю |
| ESC        | Закрыть панели настроек/логов |

### Кнопки интерфейса

| Кнопка | Описание |
|--------|----------|
| 🚀 (кнопка отправки) | Анализ страницы / Отправить запрос / Стоп |
| 📋 (копирование)     | Копировать последний ответ |
| 🗑️ (очистка)        | Очистить чат и историю |
| 📄 (контекст)       | Включить/выключить контекст страницы в чате |
| ⚙️ (настройки)     | Открыть панель настроек |
| 📖 (логи)          | Открыть панель логов |

## Настройки

### Пресеты серверов
До 3 серверов — проверяются по порядку, используется первый доступный.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| API URL  | Адрес сервера (должен заканчиваться на /v1) | http://localhost:1234/v1 |
| API Key  | Ключ авторизации | (пусто) |
| Режим извлечения | Умный / Весь контент / Только выделение | Умный |

### Модель
- **Модель** — конкретное название модели (оставьте пустым для автоопределения)
- **Использовать загруженную модель** — автоматически определять модель через GET /v1/models

### Промпты для изображений
- **🖼️ Промт изображения** — системный промпт для описания изображений
- **🔤 Промпт перевода** — системный промпт для распознавания и перевода текста

### Тема и шрифт
- Тёмная / Светлая тема
- Размер шрифта 8-16pt

## Архитектура

```
+------------------+     +------------------+     +------------------+
|  Любая страница  |---->|  Content Script  |---->|  Background JS   |
|                  |     | (extractContent) |     |  (stream + API)  |
+------------------+     +------------------+     +--------+---------+
                                                            |
+------------------+     +------------------+              |
|   Sidebar UI     |<----|  Storage (Local) |<----+---------+
|  (Чат + UI)      |     |  streamContent   |     |  AI Server      |
|                  |     |  streamProgress  |     |  (LM Studio)    |
+------------------+     +------------------+     +------------------+
```

### Компоненты

| Файл | Описание |
|------|----------|
| config.js | Централизованные настройки и константы |
| content.js | Извлечение контента со страницы (readability) |
| background.js | API запросы, streaming, retry, анализ изображений |
| sidebar.js | Логика чата, UI, управление токенами |
| sidebar.html | Разметка боковой панели |
| sidebar.css | Стили интерфейса |
| manifest.json | Конфигурация расширения |

## Безопасность

- API ключ изолирован — content script НЕ имеет доступа к ключу
- Content Security Policy — ограничение источников скриптов
- XSS защита — все пользовательские данные экранируются перед отображением в HTML
- Локальная обработка — данные не уходят на сторонние сервисы кроме вашего AI-сервера
- Работает на всех сайтах — `<all_urls>` в permissions

## Решение проблем

### Пустой контент
1. Убедитесь что страница содержит текст (не пустая)
2. На некоторых сайтах (SPA) может потребоваться полная загрузка
3. Попробуйте выделить текст вручную и отправить запрос
4. В настройках попробуйте режим "📋 Весь контент" вместо "🧠 Умный"

### AI не отвечает
1. Проверьте что AI-сервер запущен
2. Проверьте URL и API Key в настройках
3. Нажмите "🔍 Проверить доступность сервисов" в настройках
4. Для изображений убедитесь что модель поддерживает vision

### Ответ обрывается
1. Очистите историю чата (кнопка очистки)
2. Уменьшите количество предыдущих сообщений (чат автоматически обрезает историю)
3. Используйте более компактный запрос

### Изображение не обрабатывается
1. Убедитесь что модель поддерживает multimodal запросы
2. Проверьте что изображение доступно по URL
3. Если CORS блокирует — расширение автоматически попробует загрузить через background fetch

### Медленный сервер
Расширение поддерживает медленные серверы с долгой prefill:
- Таймаут до первого токена: 10 минут
- Автоматический retry при зависании стрима
- Индикатор ожидания ⏳ показывает сколько времени прошло
